## Parts description
U1 LPC4370FET100 - Main MCU.204MHz triple core(Cortex M4 + M0 * 2) and includes 80Msps ADC.  
U5 OPA4354 - Quad cmos opamp.  
U14 MCP4726A0T-E/CH - 12bit I2C DAC.
